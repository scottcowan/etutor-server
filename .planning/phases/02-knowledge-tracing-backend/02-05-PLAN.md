<objective>
TDD implementation of the `POST /v1/sessions/{id}/end` endpoint — KT-04 (D-02, D-03).

This is the canonical session-close trigger for Phase 2. It:
1. Sets `sessions.ended_at = now()`
2. Calls `update_bkt_for_session(session_id, db)` — batch BKT update for the session
3. Calls `fit_fsrs_params(child_id, db)` — per-child FSRS re-fitting
4. Returns JSON with session_id, ended_at, and count of KCs updated

| Threat ID | Category | Component | Disposition | Mitigation Plan |
